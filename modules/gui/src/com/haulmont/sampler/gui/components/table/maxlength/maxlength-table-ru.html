maxTextLength - необязательный атрибут, позволяет ограничивать количество символов в ячейке.

При этом, если разница между фактическим и допустимым количеством символов не превышает порог в 10 символов, "лишние" символы не скрываются. В данном примере только одно значение имеет длину больше maxTextLength+10 и обрезается.
