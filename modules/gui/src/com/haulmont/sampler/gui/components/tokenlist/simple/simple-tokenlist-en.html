TokenList component offers a simplified way of working with lists: instance names are listed vertically or horizontally, adding is done using drop-down list, removal – using the buttons located near each instance.

In the example the nested linesDs datasource which includes a collection of order lines within an order is defined in dsContext, as well as allLinesDs datasource containing a collection of all order lines available in the database. The TokenList component with linesList identifier displays the content of the linesDs datasource and allows changing the collection by adding instances from allLinesDs.

tokenList elements:

  • lookup − values selection component descriptor.

  • button – descriptor of the button for adding items. Can contain caption and icon attributes.
