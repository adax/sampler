LookupField можно использовать для выбора экземпляра связанной сущности. Атрибут optionsDatasource задает источник данных, содержащий список экземпляров связанной сущности.

В данном примере LookupField редактирует значение атрибута customer сущности Order, находящейся в источнике данных orderDs. В качестве списка опций используется список экземпляров сущности Customer, находящийся в источнике данных customersDs.
