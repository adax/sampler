Список опций компонента может быть задан произвольно с помощью методов setOptionsList() и setOptionsMap(), либо с помощью XML-атрибута optionsDatasource.

  • Метод setOptionsList() позволяет программно задать список опций компонента.

  • Метод setOptionsMap() позволяет задать строковые названия и значения опций по отдельности.

 Атрибут nullName позволяет задать строку, отображаемую в этом случае вместо пустой.
