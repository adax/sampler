Список опций компонента может быть задан с помощью методов setOptionsList() и setOptionsMap().

  • При использовании setOptionsList() выпадающий список будет отображать результаты преобразования каждой опции методом toString().

  • Метод setOptionsMap() позволяет явно задать строковые названия значениям опций.

Атрибут nullName позволяет задать строку, отображаемую вместо значения null.
