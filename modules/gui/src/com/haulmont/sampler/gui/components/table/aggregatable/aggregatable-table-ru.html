Атрибут aggregatable включает режим агрегации строк таблицы. Поддерживаются следующие операции:

  • SUM - сумма

  • AVG - среднее значение

  • COUNT - количество

  • MIN - минимальное значение

  • MAX - максимальное значение

Для агрегируемых колонок необходимо указать элемент aggregation с атрибутом type, задающим функцию агрегации. Агрегированные значения столбцов выводятся в дополнительной строке вверху таблицы.
