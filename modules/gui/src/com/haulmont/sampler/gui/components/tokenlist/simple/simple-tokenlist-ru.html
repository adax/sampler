Компонент TokenList представляет собой упрощенный вариант работы со списком сущностей: названия экземпляров располагаются в вертикальном или горизонтальном списке, добавление производится из выпадающего списка, удаление - с помощью кнопок, расположенных рядом с каждым экземпляром.

Здесь в элементе dsContext определен вложенный источник данных linesDs, содержащий коллекцию входящих в состав заказа строк. Кроме того, определен источник данных allLinesDs, содержащий коллекцию всех строк заказа, имеющихся в базе данных. Компонент TokenList с идентификатором linesList отображает содержимое источника данных linesDs, а также позволяет изменять эту коллекцию, добавляя в него экземпляры из источника данных allLinesDs.

Элементы tokenList:

  • lookup − описатель компонента выбора значений.

  • button − описатель кнопки добавления значений. Может содержать атрибуты caption и icon.
