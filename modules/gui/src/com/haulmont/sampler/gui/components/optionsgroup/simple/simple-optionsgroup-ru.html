OptionsGroup обеспечивает выбор из списка опций, используя группу переключателей для выбора единственного значения или группу флажков для выбора нескольких значений.

Атрибут orientation задает расположение элементов группы. По умолчанию элементы располагаются по вертикали.

В данном примере компонент используется для выбора значения перечисления (enum) для атрибута сущности.
