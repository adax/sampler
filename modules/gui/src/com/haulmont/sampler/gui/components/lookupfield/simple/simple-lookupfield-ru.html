LookupField - компонент для выбора значения из выпадающего списка. Выпадающий список реализует фильтрацию значений по мере ввода пользователя и постраничный вывод доступных значений.

В данном примере LookupField позволяет выбрать значение перечисления (enum) для атрибута сущности.

См. также: LookupPickerField | SearchPickerField
