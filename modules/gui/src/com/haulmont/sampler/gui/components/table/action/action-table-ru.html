Компонент Table реализует интерфейс ActionsHolder и может содержать как кастомные действия, так и стандартные, определенные в перечислении ListActionType.

Если действие имеет заголовок, оно отображается в контестном меню таблицы. Кроме того, действия таблицы могут быть назначены кнопкам, расположенным в любом месте экрана.

С помощью XML элемента buttonsPanel таблице можно добавить контейнер ButtonsPanel для отображения кнопок вверху таблицы.
