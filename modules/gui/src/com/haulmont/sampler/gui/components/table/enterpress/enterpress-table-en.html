The setEnterPressAction() allows to define an action executed when Enter is pressed. If such action is not defined, the table will attempt to find an appropriate one in the list of its actions in the following order:

  • The action defined by the setItemClickAction() method

  • The action assigned to the Enter key by the shortcut property

  • The edit action

  • The view action

If such action is found, and has enabled = true property, the action is executed.
