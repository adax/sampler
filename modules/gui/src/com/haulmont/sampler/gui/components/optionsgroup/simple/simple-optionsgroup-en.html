This is a component that allows you to choose from a list of options using an option group to choose a single value or a checkbox group to select several values.

The simplest case of using OptionsGroup is to select an enumeration value for an entity attribute.

In the optionsGroup component, you should specify link to a data source in the datasource attribute and a name of an entity attribute, which value should be displayed, in the property attribute.

The orientation attribute defines the orientation of group elements. By default, elements are arranged vertically. The horizontal value sets the horizontal orientation.
