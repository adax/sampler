• GeoPoint - an auxiliary component, which is not displayed on the map. This component can be used to set such map parameters as the center point, boundaries, or to create more complex map components. The object can be created using the createGeoPoint()method of the MapViewer interface.

  • Circle - a component that displays a round area. You can use the createCircle() and addCircleOverlay() methods of the MapViewer interface to create this object and put it on a map.
