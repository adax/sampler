Пользователи могут создавать условия на основе выражений JPQL, выбрав пункт Создать новое... диалога Новое условие. Кастомные условия также могут быть определены в XML экрана. Такие условия доступны пользователям в группе Специальные условия диалога Новое условие.

В данном примере условие Order items contain позволяет пользователю фильтровать заказы по продукту, содержащемуся в строке заказа. Свойства условия можно увидеть в расширенном редакторе фильтра, доступном через пункт Редактировать меню настроек фильтра.
