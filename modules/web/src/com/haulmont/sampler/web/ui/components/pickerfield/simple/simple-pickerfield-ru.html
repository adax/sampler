Компонент PickerField предназначен для отображения экземпляра сущности и выполнения действий, представляемых кнопками в правой части поля.

Компонент может быть связан с атрибутом сущности путем указания XML атрибутов datasource и property. Атрибут сущности должен быть ссылкой на другую сущность.

См. также: LookupPickerField | SearchPickerField
