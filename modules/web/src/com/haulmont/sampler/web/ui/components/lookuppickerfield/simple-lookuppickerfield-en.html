The LookupPickerField component allows a user to select an entity instance from a drop-down list and perform actions by pressing buttons on the right.

In fact LookupPickerField is a hybrid of LookupField and PickerField. Thus it has the same features except the default list of actions added when determining the component in XML: for LookupPickerField these are lookup and open.

In this example, the first field has default actions, while the second one has lookup and clear actions specified explicitly.
