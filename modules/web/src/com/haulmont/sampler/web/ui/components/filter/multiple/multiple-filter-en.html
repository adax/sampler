The Filter component enables quick data filtering by arbitrary conditions, as well as creating filters for repeated use.

In this example, the Multiple Conditions filter was created, made default and saved in the database. So it appears right after opening the screen.

You can add conditions to this filter, or reset it by selecting Reset filter in the drop-down menu next to the Search button and create a new filter. New conditions will not be saved for future use until you click Save or Save as in the settings drop-down menu on the right.
