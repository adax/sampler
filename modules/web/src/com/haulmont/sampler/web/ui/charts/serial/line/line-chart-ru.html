Последовательная диаграмма может принимать данные в виде строк с датами или временем. Даты на оси категории расположены на логических интервалах, используя любые форматы даты, какие пожелаете. Начало периода может быть выделено жирным и использовать различные форматы даты. Ваши данные могут быть в любом формате - года, месяца, дни, часы, и даже миллисекунды. Диаграмма может быть приближена или отдалена. Полосы прокрутки могут отображать приблизительный вид ваших данных, и вы можете использовать их и для прокрутки и для приближения к выбранной позиции.
