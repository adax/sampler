Последовательная дианрамма поддерживаем много типов графиков: линии, область, сглаженные линии, гистограмма, финансовый. Все типы графиков можно соединить в одну диаграмму - гистограмма с линиями, гистограмма с сглаженными линиями и т.д. Настоящее оличие всех этих графиков в одном свойстве - type, кроме финансовых, т.к. они требуют 4 значащих поля вместо одного.
