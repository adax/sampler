Компонент LookupPickerField позволяет выбирать экземпляр сущности в выпадающем списке и выполнять действия нажатием на кнопки справа.

LookupPickerField является по сути гибридом LookupField и PickerField, поэтому все описанное для этих компонентов верно и для него. Исключением является список действий по умолчанию, добавляемых при определении компонента в XML: для LookupPickerField это действия lookup и open.

В данном примере первое поле имеет действия по умолчанию, а второе - явно заданные lookup и clear.
