HtmlBoxLayout позволяет гибко задавать расположение компонентов на экране средствами HTML, а затем в формате HTML-шаблона включить его в тему приложения.

Шаблон должен содержать элементы <div> с атрибутами location, которые соответствуют атрибутам id в XML-дескрипторе компонентов экрана. Таким образом, логика описывается в XML-дескрипторе и Java-контроллере экрана, а вёрстка и сложные элементы дизайна создаются средствами разметки HTML.
