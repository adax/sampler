Контейнер CssLayout позволяет управлять размещением и стилизацией своих компонентов с помощью CSS.

Здесь приведен пример использования cssLayout в простом responsive экране. Если изменять размер экрана, то компоненты будут менять свое положение - в одну или две колонки.

Атрибут responsive указывает, что компонент должен реагировать на изменение доступного пространства. По умолчанию false.
