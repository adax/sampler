На карте могут быть размещены маркеры. Для отображения маркера, по умолчанию используется стандартная иконка сервиса карт.

В данном примере новый маркер добавляется при щелчке на карте. Щелчок на маркере открывает всплывающее окно с информацией.

Для отображения информации во всплывающем окне используется компонент InfoWindow.
